# Function: hexToBool()

> **hexToBool**(`hex_`, `opts`?): `boolean`

Decodes a hex value into a boolean.

- Docs: https://viem.sh/docs/utilities/fromHex#hextobool

## Parameters

• **hex\_**: \`0x$\{string\}\`

• **opts?**: `HexToBoolOpts`

Options.

## Returns

## Examples

```ts

const data = hexToBool('0x01')

```
