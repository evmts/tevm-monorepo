# Variable: hedera

> `const` **hedera**: `Common`

Creates a common configuration for the hedera chain.

## Description

Chain ID: 295
Chain Name: Hedera Mainnet
Default Block Explorer: https://hashscan.io/mainnet
Default RPC URL: https://mainnet.hashio.io/api
