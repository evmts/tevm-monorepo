# Variable: manta

> `const` **manta**: `Common`

Creates a common configuration for the manta chain.

## Description

Chain ID: 169
Chain Name: Manta Pacific Mainnet
Default Block Explorer: https://pacific-explorer.manta.network
Default RPC URL: https://pacific-rpc.manta.network/http
