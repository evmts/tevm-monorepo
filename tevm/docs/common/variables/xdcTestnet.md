# Variable: xdcTestnet

> `const` **xdcTestnet**: `Common`

Creates a common configuration for the xdcTestnet chain.

## Description

Chain ID: 51
Chain Name: Apothem Network
Default Block Explorer: https://apothem.blocksscan.io
Default RPC URL: https://erpc.apothem.network
