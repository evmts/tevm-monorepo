# Variable: fantom

> `const` **fantom**: `Common`

Creates a common configuration for the fantom chain.

## Description

Chain ID: 250
Chain Name: Fantom
Default Block Explorer: https://ftmscan.com
Default RPC URL: https://rpc.ankr.com/fantom

import { createMemoryClient } from 'tevm'
