# Variable: base

> `const` **base**: `Common`

Creates a common configuration for the base chain.

## Description

Chain ID: 8453
Chain Name: Base
Default Block Explorer: https://basescan.org
Default RPC URL: https://mainnet.base.org

import { createMemoryClient } from 'tevm'

import { http } from 'tevm'

})
