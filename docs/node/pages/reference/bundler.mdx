## What is the Tevm Bundler?

At build time, the bundler reads `.sol` files, runs solc on the dependency graph, extracts ABI and (for `.s.sol`) bytecode, and emits a TypeScript module exporting a [Tevm Contract](/reference/contract) instance.

For a quickstart, see [Bundler Quickstart](/getting-started/bundler).
