# @tevm/evm

The `@tevm/evm` package exposes Tevm's low-level EVM facade. It is backed by `@evmts/zevm/evm` and wrapped with Tevm-specific defaults for state management, blockchain integration, custom precompiles, predeploys, logging, profiling, and unlimited-contract-size testing.

Most applications should use `createMemoryClient`, viem actions, or JSON-RPC. Use `@tevm/evm` directly when building Tevm extensions, debuggers, profilers, or other low-level tooling.

> **Generated API Documentation**: View the full API documentation in the [evmts/tevm-monorepo/packages/evm/docs](https://github.com/evmts/tevm-monorepo/tree/main/packages/evm/docs) folder.

## Installation

```bash
npm install @tevm/evm
```

## Usage Examples

### Creating an EVM Instance

```typescript
import { createChain } from '@tevm/blockchain'
import { mainnet } from '@tevm/common'
import { createEvm } from '@tevm/evm'
import { createStateManager } from '@tevm/state'

const common = mainnet.copy()
const stateManager = createStateManager({ common })
const blockchain = await createChain({ common })

const evm = await createEvm({
  common,
  stateManager,
  blockchain,
  profiler: false,
  loggingLevel: 'warn',
})
```

### Running EVM Calls

```typescript
import { createAddress } from '@tevm/address'

const result = await evm.runCall({
  to: createAddress('0x1234567890123456789012345678901234567890'),
  caller: createAddress('0x5678901234567890123456789012345678901234'),
  data: new Uint8Array(),
  value: 1000n,
  gasLimit: 100000n,
})

console.log(result.execResult.returnValue)
console.log(result.execResult.executionGasUsed)
```

### Custom Precompiles

```typescript
import { createMemoryClient } from 'tevm'
import { createContract } from 'tevm/contract'
import { defineCall, definePrecompile } from 'tevm/precompiles'

const Calculator = createContract({
  name: 'Calculator',
  address: '0x0000000000000000000000000000000000000a11',
  humanReadableAbi: [
    'function addTwo(uint256 value) returns (uint256)',
  ],
})

const calculatorPrecompile = definePrecompile({
  contract: Calculator,
  call: defineCall(Calculator.abi, {
    addTwo: async ({ args }) => ({
      returnValue: args[0] + 2n,
      executionGasUsed: 0n,
    }),
  }),
})

const client = createMemoryClient({
  customPrecompiles: [calculatorPrecompile],
})
```

### Tracing and Logging

```typescript
const evm = await createEvm({
  common,
  stateManager,
  blockchain,
  loggingLevel: 'trace',
  profiler: true,
})

evm.events?.on('step', (step) => {
  console.log({
    pc: step.pc,
    opcode: step.opcode.name,
    gasLeft: step.gasLeft,
    depth: step.depth,
  })
})
```
