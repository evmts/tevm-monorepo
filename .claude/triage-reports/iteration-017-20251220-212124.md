### #2038: Fix TypeScript error in TevmJsonRpcRequestHandler.ts

**Problem:** TypeScript error TS2344 in `packages/actions/src/tevm-request-handler/TevmJsonRpcRequestHandler.ts` line 106. The type constraint on `JsonRpcReturnTypeFromMethod<TRequest['method']>` was failing because `anvil_enableTraces` and `anvil_mineDetailed` were present in `AnvilJsonRpcRequest` but missing from `AnvilReturnType`.

**Root Cause:** When the JSON-RPC feature parity work added new anvil methods (`anvil_enableTraces` and `anvil_mineDetailed`) to `AnvilJsonRpcRequest.ts`, the corresponding entries were not added to `AnvilReturnType.ts`. This caused the generic constraint to fail since the request type included methods that weren't mapped in the return type.

**Solution:** Added the missing method mappings to `packages/actions/src/tevm-request-handler/AnvilReturnType.ts`:
- Import: `AnvilEnableTracesJsonRpcResponse`, `AnvilMineDetailedJsonRpcResponse`
- Mapping: `anvil_enableTraces: AnvilEnableTracesJsonRpcResponse`
- Mapping: `anvil_mineDetailed: AnvilMineDetailedJsonRpcResponse`
