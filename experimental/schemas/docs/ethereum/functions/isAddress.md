# Function: isAddress()

> **isAddress**(`address`): `boolean`

Checks if a string is a valid Ethereum address.

## Parameters

▪ **address**: `string`

The address to check.

## Returns

- True if the address is valid.
[Solidity docs](https://docs.soliditylang.org/en/latest/types.html#address)
