# Function: isBytes26()

> **isBytes26**(`bytes26`): `boolean`

Type guard that returns true if the provided string is a valid Ethereum Bytes26.

## Parameters

• **bytes26**: `unknown`

## Returns

`boolean`

## Example

```ts
import { isBytes26 } from '@tevm/schemas';
isBytes26("0xff");  // true
isBytes26("0xfff"); // false
````
