# Function: isBytes19()

> **isBytes19**(`bytes19`): `boolean`

Type guard that returns true if the provided string is a valid Ethereum Bytes19.

## Parameters

• **bytes19**: `unknown`

## Returns

`boolean`

## Example

```ts

isBytes19("0xfff"); // false
````
