# Function: isBytes8()

> **isBytes8**(`bytes8`): `boolean`

Type guard that returns true if the provided string is a valid Ethereum Bytes8.

## Parameters

• **bytes8**: `unknown`

## Returns

`boolean`

## Example

```ts

isBytes8("0xff");  // true
isBytes8("0xfff"); // false
````
