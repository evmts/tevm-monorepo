# Function: isBytes17()

> **isBytes17**(`bytes17`): `boolean`

Type guard that returns true if the provided string is a valid Ethereum Bytes17.

## Parameters

• **bytes17**: `unknown`

## Returns

`boolean`

## Example

```ts

isBytes17("0xfff"); // false
````
