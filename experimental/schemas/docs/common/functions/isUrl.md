# Function: isUrl()

> **isUrl**(`value`): `boolean`

Type guard that returns true if the provided string is a valid URL.

## Parameters

• **value**: `string`

## Returns

`boolean`

## Example

```javascript
import { isUrl } from '@tevm/schemas';
isUrl('https://tevm.sh');  // true
isUrl('not a url'); // false
````
